# 如何测试 · AI 单人剧本杀

> 零配置即可跑通：**不需要任何 API Key**，缺密钥时自动用内置样例剧本「雾港庄园谋杀案」+ mock 角色发言，整套流程可玩。

## 环境要求

- **Node.js 18+**（建议 20）。检查：`node -v`

## 三步启动

```bash
npm install        # 安装依赖（结束时会自动 prisma generate）
npm run db:push    # 创建本地 SQLite 数据库（prisma/dev.db）
npm run dev        # 启动，打开 http://localhost:3000
```

> 可选：`npm run db:seed` 预置内置剧本与用户（不跑也行，首次访问会自动写入）。

## 怎么玩 / 重点测什么

打开 http://localhost:3000 后：

1. 首页点 **「内置剧本库」**（或 `/library`）→ 选「雾港庄园谋杀案」直接开局。
2. 进游戏后建议重点体验本次优化的三处：
   - **「我的剧本」** tab：查看玩法指引 + 公开身份 + 私密剧本（角色扮演模式）。
   - **「私聊」** tab：单独找一个角色密谈，AI 会直接回应你的提问。
   - **自由交流阶段**：AI 角色会在公共大厅彼此讨论；也可点输入框上方 **「让大家讨论一轮」**，或随时自己插话。
3. 跟随 DM 推进阶段 → 投票指凶 → 复盘揭秘。

## 接入真实大模型（可选，效果更好）

包内已附带一个 **空的 `.env`**（所有 key 留空，默认走 mock）。直接编辑它，填入你自己的 key 即可：

```bash
# 打开根目录的 .env，任填其一：
#   STEP_API_KEY=...        （Step 阶跃星辰）
#   ANTHROPIC_API_KEY=sk-ant-...   （Anthropic Claude）
```

- Step 阶跃星辰：https://platform.stepfun.com/
- Anthropic Claude：填 `ANTHROPIC_API_KEY=sk-ant-...`

填好后重启 `npm run dev` 即可，无需改代码。

## 常见问题

- **端口 3000 被占用**：先关掉占用进程，或 `PORT=3001 npm run dev`。
- **页面报数据库错误**：确认已执行 `npm run db:push`。
- 更多说明见 `README.md`。
